# ⚙️ CPU Scheduling Visualizer

A web-based simulator for visualizing CPU scheduling algorithms with Gantt charts and performance comparison.

---

## 🚀 Features

- ⚡ Algorithms Implemented:
  - FCFS (First Come First Serve)
  - SJF (Shortest Job First)
  - Round Robin
  - SRTF (Shortest Remaining Time First)

- 📊 Displays:
  - Completion Time (CT)
  - Turnaround Time (TAT)
  - Waiting Time (WT)

- 📈 Visualization:
  - Gantt Chart representation
  - Average WT & TAT calculation
  - Algorithm comparison table

- 🎨 UI:
  - Interactive input system
  - Modern dark-themed interface

---

## 🧠 Concepts Used

| Concept            | Implementation |
|-------------------|--------------|
| Scheduling Algorithms | FCFS, SJF, RR, SRTF |
| Data Structures   | Arrays, Queues |
| Visualization     | Dynamic Gantt Chart |
| Algorithm Analysis | WT, TAT, CT calculations |
