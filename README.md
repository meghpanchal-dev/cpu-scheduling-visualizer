# ⚙️ CPU Scheduling Visualizer

A web-based simulator for visualizing CPU scheduling algorithms with Gantt charts and performance comparison.

---

## 🚀 Features

- FCFS, SJF, Round Robin, SRTF algorithms  
- Gantt Chart visualization  
- Waiting Time (WT) and Turnaround Time (TAT) calculation  
- Algorithm comparison table  
- Interactive UI  

---

## 🧠 Concepts Used

- CPU Scheduling Algorithms  
- Data Structures (Queues, Arrays)  
- Algorithm Analysis  

---
## ▶️ How to Run

Open `index.html` in browser
